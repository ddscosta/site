1. Apagar os pontos e histórico sem confirmação.

    2. Para apagar todos os pontos exibidos atualmente(acertos, erros e pulos) ... e todo o hitórico armazenado, digite a chave '?del' no final do endereço da pagina desejada.

    Exemplo: .../adicao1.html?del

  2. Imprimir o histórico sem os calculos do aluno.

    3. Por padrão, o histórico armazena as operações que o usuário acertou, errou ou pulou e seus respectivos resultados. Para imprimir estes dados mas sem os resultados, digite a chave e valor '?his=1' no final do endereço da pagina de impressão do histórico.

    Exemplo: .../histadi.html?his=1

  3. Mudar nome do Aluno na impressão dos resultados

    4. Se desejar o nome do usuário na página do histórico a ser impresso insira-o no Menu principal no campo Aluno. Após digitar o nome clique no menu novamente para que seja salvo.

    1. Se o campo nome não mudar, nada acontece.
    2. Se o campo nome mudar uma mensagem de aviso será mostrada.
    3. Se a mensagem for cancelada os dados não serão apagados.

    Obs.:Para que não haja mistura dos dados do usuário anterior com o atual, o campo aluno deve ser trocado antes de iniciar os cálculos. Além disso, os pontos também devem ser zerados separadamente antes do inicio de tudo. Caso contrário haverá divergências entre a quantidade de pontos exibidos e os dados do histórico.
